# 三农新闻日报系统 — 业务流程图

> 最后更新：2026-05-30

---

## 数据存储位置

```
C:\学习\AI文件\data\
└── 2026-05-30/                    ← 按日期自动创建
    ├── raw/                        ← 原始抓取数据（按来源拆分）
    │   ├── people_daily.json       ← 人民日报抓的原文
    │   ├── xinhua.json             ← 新华社
    │   ├── farmers_daily.json
    │   └── ...
    ├── filtered.json               ← AI 第1层：筛选后的高价值新闻
    ├── classified.json             ← AI 分类+摘要+评分
    ├── daily_brief.md              ← 最终日报（Markdown，可直接阅读）
    ├── errors.json                 ← 抓取失败的源及原因
    └── metadata.json               ← 运行统计（耗时、成功率等）

archive.db                          ← SQLite 去重库（已抓过的 URL 不再重复抓）
```

## 运行命令速查

| 命令 | 说明 |
|------|------|
| `python src/main.py --dry-run` | 只测试爬虫，不调 AI |
| `python src/main.py --source 人民日报` | 只爬单个源 |
| `python src/main.py --no-push` | 完整运行，不发微信 |
| `python src/main.py` | 完整运行（含推送） |
| `python src/main.py --deep "高标准农田"` | 深度专题分析 |
| `python -m pytest tests/ -v` | 运行全部测试 |

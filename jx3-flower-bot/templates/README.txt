剑网三 种花脚本 - 模板图片截取指南
====================================

脚本依赖模板图片来识别游戏界面元素，你需要自己截取这些图片。

准备工作
--------
1. 游戏设置为「无边框窗口」模式（系统设置 → 显示设置）
2. 分辨率推荐 1920x1080（其他分辨率模板仍然可用，但识别率可能略降）
3. 准备截图工具：Windows 自带的截图工具或 Snipaste 均可

模板清单
--------

1. empty_pot.png  — 空花盆
   截图时机：找一个没有种植的花盆，正对着花盆截图
   要求：花盆在画面中央，清晰可见
   推荐尺寸：约 60x60 像素

2. mature_flower.png  — 成熟花朵
   截图时机：花盆里的花已经长好（可收获状态）
   要求：花朵完整清晰
   推荐尺寸：约 60x60 像素

3. plant_btn.png  — 「播种」按钮
   截图时机：点击空花盆后弹出的 UI 界面中的播种按钮
   要求：只截取按钮图标部分，不要包含背景文字
   推荐尺寸：约 40x40 像素

4. harvest_btn.png  — 「收获」按钮
   截图时机：点击成熟花盆后弹出的收获按钮
   要求：只截取按钮图标部分
   推荐尺寸：约 40x40 像素

5. water_btn.png  — 「浇水」按钮（可选）
   截图时机：点击生长中的花盆后弹出的浇水按钮
   要求：只截取按钮图标部分
   推荐尺寸：约 40x40 像素

6. confirm_btn.png  — 「确认」按钮（可选）
   截图时机：游戏中各种确认弹窗的确定按钮
   作用：辅助关闭弹窗

截取技巧
--------
- 使用截图工具时，按住 Alt 键可以只截取游戏窗口
- 截取后裁剪到只保留关键元素
- 模板文件越小越好（几十像素即可），太大了匹配反而慢
- 如果同一个元素在不同状态下颜色不同，建议分别截图

快速验证
--------
截好模板后运行：
    python main.py

脚本启动时会自动加载 templates/ 目录下的所有模板并打印加载状态。
